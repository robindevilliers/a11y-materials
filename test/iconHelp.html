We have used Materialize in order to implement the look and feel of this website.

Consequently, the implementation of icons is provided by Materialize.

Materialize Icons

In order to configure an icon, click on the link above, and the Materialize website will open. If you browse downwards in that page, you will find an extensive list of icons with their names supplied. Simply replace the name on the left with whichever name you prefer.

If you require an icon that is not included in the list, please consult the web developers.
